Backup & Restore

Create Backup

Create a complete backup of the YouthConnect database. This will download a .SQL file containing all current data.

Restore Database

Warning: Restoring a backup will overwrite all current data in the database. This action cannot be undone.

i Important Information

  • Backups are stored as SQL files and can be imported into any MySQL database
  • Regular backups are recommended to prevent data loss
  • Always test restore operations in a development environment first
  • Keep backup files in a secure location
  • Backup files contain sensitive information - handle with care
